Weather Application

Find the weather for any city

I built a weather application using the Open Weather API and Flask to run it on my web browser. This was a project in a long list of projects for beginners who are learning how to use Flask. I followed along with the tutorial by Pretty Printed in order to properly learn how to use the Open Weather API, because I had never used an API before.

But, hey, it works! I had a lot of fun working with this application and learning how to use API's. They are extremely useful and I highly recommend messing around with some.

In this application, you can type the name of a city into the text input box, click the submit/'Find Weather' button, and it will route itself through the API to find the weather of the city you type. It also has a Delete feature, so you don't have to worry about filling up the page with too many cities' weather info.
