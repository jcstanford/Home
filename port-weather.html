Weather Application

A web application

I built a weather forecasting application with the aid of the "Pretty Printed" Youtube Channel. This app has a foundation of HTML, CSS, and Flask and utilizes the OpenWeather API to get weather data.

Building this application was my first time working with an API, so I was able to learn a lot while building a fun little tool that people use everyday. Flask is a documentation in Python that makes it possible to build web sites and web apps.
