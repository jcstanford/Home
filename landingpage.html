Landing Page Design

for a generic pizza company

I made an unoriginal landing page for a fake pizza company. This page was created as a part of completing a web design certification through freecodecamp.org. Designing this allowed me to test my knowledge of HTML and CSS, which is where I first started learning how to code.

This is a basic web page that does not have anything in terms of user interface (UI), so there are menu items and contact info, but nothing really practical. I would like to expand on this and potentially create an avenue for a "create your own pizza" or an Order Online feature, since that is included with every other pizza restaurant these days.

The page is meant to show what I can do for someone looking to built a web landing page for their business, but it is just a start. Since designing this, I have found there are other ways for building websites.
